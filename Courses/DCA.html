Diploma in Computer Applications (DCA):
DCA is a diploma course that introduces students to the basics of computer applications and programming languages. It covers topics such as computer fundamentals, operating systems, programming languages (such as C, C++, Java), software applications (like Microsoft Office), database management, and internet technologies. DCA programs are typically designed for individuals who want to gain practical knowledge and skills in computer applications and software development.
  1. Computer fundamentals:
    • Computer Hardware
    • Computer Software
    • Data Representation
    • Computer Networking
    • Algorithms and Programming
    • Computer Security
    • Computer Ethics and Privacy
    • Computer History and Future Trends
  2. Operating Systems:
    • MS Dos
    • Windows XP
    • Windows 7
  3. MS Office
    • Excel
    • Power Point
    • Word
  4. C++
  5. Java
